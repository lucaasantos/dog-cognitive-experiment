# 🧠 Dog Cognitive Experiment

Experimental adaptive cognitive AI system focused on:
- persistent memory
- behavioral adaptation
- belief evolution
- machine learning classification
- real-time cognitive observability

The project evolved from a symbolic rule-based system into a hybrid cognitive architecture combining:

- Symbolic reasoning
- CountVectorizer
- Naive Bayes
- Persistent memory
- Confidence modeling
- Live cognitive visualization

### V1 — Symbolic Cognitive System
- Rule-based behavioral adaptation
- Emotional weight simulation
- Persistent JSON memory
- Belief confidence system

### V2 — Hybrid ML Cognition
- CountVectorizer integration
- Multinomial Naive Bayes
- Dataset-based learning
- Probabilistic classification

### V3 — Cognitive Observability Dashboard
- PySide6 futuristic interface
- Real-time learning visualization
- Live cognitive stream
- Explainable ML panels
- Dynamic state monitoring

##  Core Concepts

- Adaptive AI Systems

- Hybrid Symbolic + ML Cognition
